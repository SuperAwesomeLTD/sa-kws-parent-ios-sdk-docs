Kids Web Services Parents Android SDK¶

The Kids Web Services platform provides a set of compliance-as-a-service tools to create a kid-safe, COPPA-compliant framework for kids sites and apps. Focus on delivering awesome content, whilst KWS manages the legal complexity of data-collection for you. With simplicity built into its core, you will be able to effortlessly integrate Kids Web Services within your current technology.
